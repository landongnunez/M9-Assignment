# Module 09: GlobalTech Quarterly Sales Analysis

This repository contains an introductory data analysis project using Pandas to analyze quarterly sales performance for a global technology company.

## Files Included
- `module09-assignment.py`: Python script that processes a simulated CSV dataset to generate a quarterly sales report.

## What I Practiced
- **Data Exploration:** Using `.info()`, `.describe()`, and `.head()` to understand dataset structure.
- **Filtering & Selection:** Isolating specific regions (North America), high-volume sales, and promotional performance.
- **Advanced Grouping:** Aggregating total revenue by region and calculating average unit prices by product category.
- **Promotion Analysis:** Comparing average sales and total revenue between promoted and non-promoted items.
- **Business Reporting:** Summarizing findings into a clear performance report with actionable recommendations.
